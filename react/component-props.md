# ComponentProps<T>

Used to automatically infer props type of a [[react-js]] component especially for use with [[hoc]]s, etc.

```ts
export type ComponentProps<T> = T extends
  | React.ComponentType<infer P>
  | React.Component<infer P>
  ? P
  : never;
```
